# Icons Placeholder

For now, the extension will work without custom icons (Chrome will use default).

To add custom icons later:
1. Create icon16.png (16x16)
2. Create icon48.png (48x48)
3. Create icon128.png (128x128)

You can use any image editor or online tool like:
- https://www.favicon-generator.org/
